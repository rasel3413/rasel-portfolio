### Customization

#### Colors
Update CSS variables in `assets/css/main.css`:
```css
:root {
  --accent-color: #3b82f6;        /* Primary brand color */
  --background-color: #0a0a0a;    /* Main background */
  --surface-color: #1f2937;       /* Card/component backgrounds */
}
```

#### Content
Edit the following files:
- `index.html` - Main content and structure
- `portfolio-details-*.html` - Individual project details
- `assets/css/main.css` - Styling and animations
- `assets/js/main.js` - Interactive features

#### Adding Projects
1. Add project images to `assets/img/portfolio/`
2. Create new portfolio detail page
3. Update portfolio grid in `index.html`
4. Add appropriate filters and categories
