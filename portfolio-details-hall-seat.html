Hall Seat Management System

Hall Seat Management System is a comprehensive Django-based web application designed to streamline university hostel operations. The system efficiently handles room allocation, manages student complaints, and provides robust tools for effective room management.

Built during my university years to address real-world hostel management challenges, this project demonstrates my ability to create practical solutions using modern web technologies and database design principles.

Key Features:

  • Automated room allocation system
  • Student complaint management
  • Room availability tracking
  • Admin dashboard for management
  • Student profile management
  • Reporting and analytics

Technology Stack:

Django Python Bootstrap PostgreSQL HTML/CSS JavaScript
