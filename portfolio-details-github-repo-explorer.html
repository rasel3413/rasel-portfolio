GitHub Repository Explorer

GitHub Repository Explorer is a Flutter application designed to explore GitHub repositories seamlessly. It leverages the Provider package for efficient state management, integrates HTTP requests to the GitHub API for fetching real-time repository data, and supports offline browsing through local caching. As a self-initiated project, I built this app to enhance my skills in Flutter development, API integration, and offline functionality, delivering a user-friendly experience for developers and GitHub enthusiasts.
