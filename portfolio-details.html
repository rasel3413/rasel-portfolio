Restrogreen POS App

Restrogreen POS App is an Android Studio-based point-of-sale solution designed for restaurant management. Built with Kotlin and integrated with a Laravel backend, this app streamlines operations with features like order processing, inventory tracking, payment integrations, and CRM tools.

As a Software Engineer L1 at Fronture Technologies Ltd., I contributed to its development and maintenance, enhancing stability and adding key functionalities to improve the user experience for restaurant staff and owners.

Key Features:

  • Order processing and management
  • Inventory tracking system
  • Payment integration
  • Customer relationship management
  • Real-time reporting
  • Multi-location support

Technology Stack:

Kotlin Android Studio Laravel MySQL REST API Material Design
