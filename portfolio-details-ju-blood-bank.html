JU Blood Bank

JU Blood Bank is a comprehensive Android application developed to facilitate blood donation within the Jahangirnagar University community. Built with Kotlin and powered by Firebase, this app serves as a vital bridge between blood donors and recipients in emergency situations.

The application features real-time donor search capabilities, user authentication, comprehensive donor profiles, and an intuitive interface designed to ensure accessibility and ease of use during critical moments.

Key Features:

  • Blood donor search by group and location
  • Real-time availability updates
  • User authentication and profiles
  • Emergency contact integration
  • Donor history and statistics
  • Offline data synchronization

Technology Stack:

Kotlin Android Studio Firebase Room Database Material Design Google Maps API
